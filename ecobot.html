EcoBot ( Simulasi )

Antarmuka chatbot sederhana untuk tanya jawab seputar aksi ramah lingkungan. Semua berjalan frontend-only (tanpa backend/API) agar sesuai aturan lomba.

Contoh Pertanyaan :

  • Bagaimana cara memilah sampah organik & anorganik?
  • Apa manfaat menanam pohon di sekolah?
  • Tips mengurangi plastik saat jajan?
  • Bagaimana cara hemat listrik di rumah?
  • Di mana cari bank sampah terdekat?

Demo :

  • EcoBot menggunakan pencocokan kata kunci sederhana (rule-based) di sisi klien. Ini mematuhi batasan lomba: tanpa framework JS tambahan & tanpa backend.
Privasi: Tidak ada data yang dikirim ke server. Pesan hanya tampil di browser ini sebagai mockup.
